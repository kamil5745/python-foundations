# Task Tracker

Консольный трекер задач на Python.

## Возможности

- Добавление задач
- Просмотр задач
- Удаление задач
- Отметка задачи выполненной
- Поиск задач
- Сохранение в JSON

## Установка

python -m venv .venv
pip install -r requirements.txt

## Запуск

cd task_tracker_project
python main.py

## Структура проекта

main.py
models.py
storage.py
menu.py
validators.py
exceptions.py
